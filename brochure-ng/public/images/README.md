## Image Optimization Tips

1. **Resolution**: Use high-resolution images (at least 300 DPI for print version)
2. **Color Profile**: sRGB color space for web, CMYK for professional printing
3. **File Size**: Optimize for web (aim for <500KB per image) without quality loss
4. **Aspect Ratios**: Match the specified dimensions exactly to avoid distortion
5. **Formats**: 
   - Use PNG for logo (transparency support)
   - Use JPG for photos (better compression)
